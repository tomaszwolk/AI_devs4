## How to use

uv run main.py
if model asks for advice you can tell him what to do or just press enter and he will get DEFAULT_CONTINUATION_HINT
before model will try to execute code he will ask for permission, you can agree (y), disagree (n) or edit (edycja)

by using bonus.py you can send requests to API
uv run bonus.py ENDPOINT QUERY
ENDPOINT - verify (to check answer), maps, books, wehicles, toolsearch
