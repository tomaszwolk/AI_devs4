Poniżej przedstawiono najczęstsze kody błędów wyświetlane na terminalu stacyjnym:

| Kod | Znaczenie | Akcja operatora |
|---|---|---|
| E-001 | Identyfikator nadawcy nieczytelny | Poproś o ponowne przyłożenie karty |
| E-002 | Identyfikator nadawcy wygasły | Odsyłaj nadawcę do Punktu Rejestracyjnego |
| E-003 | Nadawca zablokowany | Odmów przyjęcia, nie informuj o przyczynie |
| E-004 | Odbiorca nie istnieje w systemie | Weryfikuj numer identyfikatora |
| E-005 | Odbiorca zablokowany | Odmów przyjęcia, nie informuj o przyczynie |
| E-010 | Awaria modułu wagowego AK | Zgłoś incydent, wstrzymaj przyjmowanie |
| E-011 | Awaria modułu wymiarowego AK | Zgłoś incydent, wstrzymaj przyjmowanie |
| E-012 | Awaria modułu rentgenowskiego AK | Zgłoś incydent, wstrzymaj przyjmowanie, EWAKUACJA strefy AK |
| E-013 | Awaria modułu spektrometrycznego AK | Zgłoś incydent, wstrzymaj przyjmowanie |
| E-014 | Awaria modułu plombującego AK | Zgłoś incydent, wstrzymaj przyjmowanie |
| E-020 | Brak łączności z Systemem | Przejdź w tryb oczekiwania, NIE przyjmuj przesyłek |
| E-021 | Łączność niestabilna | Kontynuuj z podwyższoną ostrożnością |
| E-030 | Brak wolnych wagonów | Informuj nadawców o opóźnieniu |
| E-040 | Trasa zablokowana | Informuj nadawców, że przesyłki na danej trasie nie są przyjmowane |
| E-099 | Błąd krytyczny | NATYCHMIAST powiadom Dyspozytora Lokalnego |
