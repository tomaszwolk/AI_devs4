# AI_devs4

# Overview

These are my solutions to the AIDevs4. For details, see https://aidevs.pl.
To fully understand the code, you need to have access to the training.

The solutions are intentionally simplistic. The goal of this training
is to gain experience with OpenAI and other AI tools, not to show off python skills.

# Installation

1. Setup the python environment

```
uv sync
```

2. Create .env file using .env.example

# Usage:

Go to folder with lesson and run `main.py`. For example:

`cd S05E05`
`uv run main.py`
